#Event-App 

Build with 

 1. Angular 1.x
 2. Webpack
 3. ES-2015 (next generation javascript)
 4. Node (for node-modules)
 5. Karma (test runner)
 6. Jasmine (unit test)
 7. HTML5
 8. CSS

System Requirement  

node
webpack (global)
server ( I used live-server )

 Clone project 
`git clone repository`

Get latest code
  
`git pull`

Get all dependencies 

    npm install

 Build application
 

    npm run build
Test application

    npm run test

 Serve application to browser

    live-server dist
